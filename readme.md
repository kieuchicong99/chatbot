## Chatbot with nltk, tflearn, tensorflow, numpy

### Cài đặt môi trường 
- Cài đặc docker và docker-compose
- Các lệnh cần thiết 
    ```bash
    dockerd #start docker
    docker-compose build #build image
    docker-compose run chatbot bash -c "python test.py" # trainning and test

    ```
